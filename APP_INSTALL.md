## Installation Methods

### Method 1: TestFlight (iOS - Recommended)
1. Install TestFlight from App Store
2. Scan the QR code above or click the link
3. Accept the invitation
4. Install Fit Fusion

### Method 2: Expo Go (iOS & Android - Development)
1. Install Expo Go from App Store or Play Store
2. Scan this QR code in Expo Go:

![Expo Go QR Code](https://api.qrserver.com/v1/create-qr-code/?size=300x300&data=exp://your-expo-url)

### Method 3: Direct APK (Android)
1. Enable "Install from Unknown Sources" in Settings
2. Scan the Android QR code above
3. Download and install the APK

### Method 4: Build from Source
```bash
# Clone the repository
git clone https://github.com/yash1262/Fit-Fusion-mobile-app.git
cd Fit-Fusion-mobile-app/mobile

# Install dependencies
npm install

# Run on iOS
npm run ios

# Run on Android
npm run android
```
