üìã Instructions to Generate favicon.ico

Option 1: Online Tool (Easiest)

  1. Go to favicon.io/favicon-converter
  2. Upload your public/favicon.svg file
  3. Click "Download" to get favicon.ico
  4. Replace public/favicon.ico with the downloaded file

Option 2: Using ImageMagick (Command Line)

  1. Install ImageMagick: brew install imagemagick (Mac) or sudo apt install imagemagick (Linux)
  2. Run: convert public/favicon.svg -resize 32x32 public/favicon.ico

Option 3: Using Node.js Script

  1. Install sharp: npm install sharp
  2. Run: node generate-logo-pngs.js
  3. This will generate all PNG logos including favicon
